PROJECT OVERVIEW

в”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђ

MiniIDEv04 is a WPF-based mini IDE shell built in .NET / C#.

It uses SQLite (sqlite-net-pcl) as the single source of truth for all runtime
state вЂ” panels, notes, settings, git log, drop log.

UI is driven by CommunityToolkit.Mvvm source generators.

Telerik controls (Windows11 theme) are used for UI components.

GitHub repo: https://github.com/RBoxGitHub/MiniIDEv04
