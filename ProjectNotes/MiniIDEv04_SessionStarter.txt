PROJECT OVERVIEW
────────────────
MiniIDEv04 is a WPF-based mini IDE shell built in .NET / C#.
It uses SQLite (sqlite-net-pcl) as the single source of truth for all runtime
state — panels, notes, settings, git log, drop log.
UI is driven by CommunityToolkit.Mvvm source generators.
Telerik controls (Windows11 theme) are used for UI components.
GitHub repo: https://github.com/RBoxGitHub/MiniIDEv04

KEY CONVENTIONS
───────────────
- Namespace: MiniIDEv04 (always match project name)
- All data access via repository classes ONLY — never SQLiteConnection directly in UI
- All new panels: implement IDraggablePanel, register in PanelControlFactory, add seed in ProjectDatabase
- All new DB tables: add CreateTable<T>() call in ProjectDatabase.Initialize()
- DB path: ProjectNotes/miniIDE.db (relative to AppContext.BaseDirectory)
- CommunityToolkit.Mvvm: use [ObservableProperty] and [RelayCommand] source generators
- ProjectRootPath (in sys_AppSettings): D:\GrokCryptoTrack\Production-Claude\MiniIDE-WorkFolder\MiniIDEv04
