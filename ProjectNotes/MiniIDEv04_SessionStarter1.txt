PROJECT OVERVIEW

────────────────

MiniIDEv04 is a WPF-based mini IDE shell built in .NET / C#.

It uses SQLite (sqlite-net-pcl) as the single source of truth for all runtime
state — panels, notes, settings, git log, drop log.

UI is driven by CommunityToolkit.Mvvm source generators.

Telerik controls (Windows11 theme) are used for UI components.

GitHub repo: https://github.com/RBoxGitHub/MiniIDEv04

KEY CONVENTIONS

───────────────

- Namespace: MiniIDEv04 (always match project name)
- All data access via repository classes ONLY — never SQLiteConnection directly in UI
- All new panels: implement IDraggablePanel, register in PanelControlFactory, add seed in ProjectDatabase
- All new DB tables: add CreateTable<T>() call in ProjectDatabase.Initialize()
- DB path: ProjectNotes/miniIDE.db (relative to AppContext.BaseDirectory)
- CommunityToolkit.Mvvm: use [ObservableProperty] and [RelayCommand] source generators
- [RelayCommand] requires PRIVATE methods — public methods must be called directly
- No Telerik controls in modal/popup windows — plain WPF only (black window issue)
- EnableDefaultItems=False in .csproj — every new file MUST be added manually
- ProjectRootPath (in sys_AppSettings): D:\GrokCryptoTrack\Production-Claude\MiniIDE-WorkFolder\MiniIDEv04

RADSYNTAXEDITOR KNOWN API (Telerik 2026.1.211)

───────────────────────────────────────────────

- XamlEditor.Document.Insert(int offset, string text)  ← confirmed working
- XamlEditor.Document.TextContentChanged  ← confirmed working (fires on every edit)
- XamlEditor.Document.PreviewTextContentChanged  ← event
- XamlEditor.Document.CurrentSnapshot.GetText()  ← confirmed working (gets full text content)
- XamlEditor.Document.ReplaceNextMatch(string, int, string, bool, bool)  ← search/replace only
- XamlEditor.Document.ToString()  ← returns TYPE NAME, not content — do NOT use for text
- XamlEditor.Document.Inserted  ← does NOT exist on TextDocument
- XamlEditor.Document.Deleted   ← does NOT exist on TextDocument
- XamlEditor.TextChanged  ← does NOT exist
- XamlEditor.CaretOffset  ← does NOT exist
- XamlEditor.Document.Load(stream)  ← does NOT exist
- XamlEditor.Document.Replace(int, int, string)  ← does NOT exist

EDITOR TEXT READ PATTERN (confirmed working):

  // Wire once in OnContentRendered:
  XamlEditor.Document.TextContentChanged += (s, e) =>
  {
      if (s is Telerik.Windows.SyntaxEditor.Core.Text.TextDocument doc)
          _xamlEditorText = doc.CurrentSnapshot.GetText();
  };
  // Provide to ViewModel via delegate:
  _vm.XamlContentProvider = () => _xamlEditorText;
